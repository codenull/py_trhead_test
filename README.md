# py_thread_test
Сравнение многопоточности и использования процессов в python3.

# Задача
* Создать одномерный длинный массив.
* Заполняют этот массив единицами.
* Проходить по всему массиву и для каждой ячейки считают новое значение по формуле:
```
arr[i] = (arr[i] * sin(0.2 + i / 5) * cos(0.2 + i / 5) * cos(0.4 + i / 2))
```
* Заполнение массива по формуле можно выполнять в несколько потоков.
