# py_trhead_test
Сравнение многопоточности и многопроцессности в python3.

# Задача
* Создать одномерный длинный массив, например:
* Заполняют этот массив единицами;
* Проходят по всему массиву и для каждой ячейки считают новое значение по формуле:
```
arr[i] = (arr[i] * Math.sin(0.2f + i / 5) * Math.cos(0.2f + i / 5) * Math.cos(0.4f + i / 2))
```
* Заполнение массива по формуле выполнять в несколько потоков.
